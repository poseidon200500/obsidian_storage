# **Алгоритмы на графах**  
*Методы анализа и обработки графовых структур для решения задач на связность, кратчайшие пути и оптимизацию в сетях.*

#### **1. [[Обход графов]]**  
*Фундаментальные алгоритмы посещения всех вершин графа.*  
- Поиск в глубину (DFS)
- Поиск в ширину (BFS)
- Топологическая сортировка
- Поиск компонент связности
- Обход с метками времени

#### **2. [[Кратчайшие пути]]**  
*Алгоритмы нахождения минимального расстояния между вершинами.*  
- Алгоритм Дейкстры
- Алгоритм Беллмана-Форда
- Алгоритм Флойда-Уоршелла
- A* поиск (A-star)
- Сравнение алгоритмов

#### **3. [[Минимальное остовное дерево]]**  
*Построение дерева минимального веса, соединяющего все вершины.*  
- Алгоритм Прима
- Алгоритм Крускала
- Проверка критериев оптимальности
- Применение в сетевом проектировании

#### **4. [[Потоки в сетях]]**  
*Алгоритмы для задач максимального потока и минимального разреза.*  
- Алгоритм Форда-Фалкерсона
- Алгоритм Эдмондса-Карпа
- Алгоритм Диница
- Паросочетания в двудольных графах
- Задачи минимизации стоимости

#### **5. [[Связность и мосты]]**  
*Анализ структурной устойчивости графов.*  
- Поиск мостов и точек сочленения
- Компоненты сильной связности (алгоритм Косарайю)
- 2-связные компоненты
- Алгоритм Тарьяна для поиска компонент
- Применение в сетевой надежности

#### **6. [[Специальные графы]]**  
*Алгоритмы для работы с графами особых типов.*  
- Деревья и их свойства
- Эйлеровы и гамильтоновы пути
- Планарные графы
- Графы интервалов
- Алгоритмы на ориентированных графах
