# **Жадные алгоритмы**  
*Алгоритмы, принимающие локально оптимальные решения на каждом шаге в надежде найти глобально оптимальное решение. Эффективны для задач с определенными свойствами.*

#### **1. [[Принципы жадных алгоритмов]]**  
*Фундаментальные концепции и условия применимости жадных подходов.*  
- Принцип жадного выбора
- Оптимальная подструктура
- Доказательство корректности
- Когда жадный подход не работает
- Сравнение с ДП

#### **2. [[Интервальные задачи]]**  
*Задачи, связанные с работой с интервалами и временными промежутками.*  
- Выбор максимального числа непересекающихся интервалов
- Покрытие точек интервалами
- Объединение интервалов
- Расписание мероприятий
- Задачи на временные отрезки

#### **3. [[Задачи на графах]]**  
*Применение жадных алгоритмов для решения графовых задач.*  
- Алгоритм Крускала для MST
- Алгоритм Прима для MST
- Раскраска графов
- Задачи на пути в графах
- Покрытие вершин/рёбер

#### **4. [[Кодирование и сжатие]]**  
*Оптимальные методы кодирования информации.*  
- Алгоритм Хаффмана
- Префиксные коды
- Адаптивное кодирование
- Оптимальное слияние последовательностей
- Приоритетные очереди в жадных алгоритмах

#### **5. [[Задачи распределения]]**  
*Оптимальное распределение ресурсов и задач.*  
- Задача о рюкзаке (жадный вариант)
- Распределение загрузки процессоров
- Балансировка нагрузки
- Задачи планирования
- Распределение с ограничениями

#### **6. [[Классические задачи]]**  
*Известные проблемы, эффективно решаемые жадными методами.*  
- Задача о выборе заявок
- Размен монет (жадный подход)
- Задача о станциях
- Максимальный отрезок суммы
- Поиск максимума произведения
