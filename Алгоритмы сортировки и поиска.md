# **Алгоритмы сортировки и поиска**  
*Фундаментальные методы упорядочивания данных и эффективного поиска элементов. Составляют основу для решения более сложных алгоритмических задач.*

#### **1. [[Сравнительные сортировки]]**  
*Алгоритмы, основанные на сравнении элементов между собой.*  
- Быстрая сортировка (Quicksort)
- Сортировка слиянием (Mergesort)
- Пирамидальная сортировка (Heapsort)
- Сортировка вставками (Insertion Sort)
- Сортировка пузырьком (Bubble Sort)

#### **2. [[Несравнительные сортировки]]**  
*Алгоритмы, не использующие сравнение элементов, а основанные на их свойствах.*  
- Сортировка подсчетом (Counting Sort)
- Поразрядная сортировка (Radix Sort)
- Блочная сортировка (Bucket Sort)
- Ограничения и область применения

#### **3. [[Бинарный поиск]]**  
*Эффективный алгоритм поиска в отсортированных массивах.*  
- Классический бинарный поиск
- Поиск левой и правой границы
- Поиск в повернутом массиве
- Поиск в бесконечном массиве
- Применение в задачах оптимизации

#### **4. [[Поиск в строках]]**  
*Алгоритмы для поиска подстрок и шаблонов в тексте.*  
- Наивный алгоритм поиска
- Алгоритм Кнута-Морриса-Пратта (KMP)
- Алгоритм Бойера-Мура
- Поиск с использованием хешей
- Регулярные выражения

#### **5. [[Алгоритмы выбора]]**  
*Методы нахождения k-го порядкового элемента в массиве.*  
- Медиана медиан (BFPRT)
- Quickselect алгоритм
- Поиск k минимальных/максимальных элементов
- Статистические алгоритмы

#### **6. [[Специализированные поиски]]**  
*Алгоритмы для работы с особыми структурами данных.*  
- Поиск в матрице
- Интерполяционный поиск
- Экспоненциальный поиск
- Тернарный поиск
- Поиск в потоке данных
