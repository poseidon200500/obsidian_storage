# **Бинарное дерево поиска**  
*Структура данных для эффективного хранения и поиска упорядоченных данных. Основные операции выполняются за логарифмическое время в сбалансированных деревьях.*

#### **1. [[Основы BST]]**  
*Фундаментальные свойства и операции бинарных деревьев поиска.*  
- Определение и свойства BST
- Базовая структура узла
- Проверка валидности BST
- Высота и балансировка
- Основные инварианты

#### **2. [[Операции в BST]]**  
*Стандартные операции работы с бинарными деревьями поиска.*  
- Поиск элемента
- Вставка нового узла
- Удаление узла
- Нахождение минимума/максимума
- Поиск предшественника и преемника

#### **3. [[Обходы BST]]**  
*Методы traversal для обработки всех узлов дерева.*  
- In-order обход (симметричный)
- Pre-order обход (прямой)
- Post-order обход (обратный)
- Level-order обход (по уровням)
- Применение различных обходов

#### **4. [[Сбалансированные BST]]**  
*Модификации BST для поддержания логарифмической высоты.*  
- AVL деревья
- Красно-черные деревья
- B-деревья и B+ деревья
- Splay деревья
- Сравнение различных подходов

#### **5. [[Оптимизации BST]]**  
*Методы улучшения производительности BST.*  
- Кэширование часто используемых элементов
- Bulk-операции
- Parallel operations
- Memory layout оптимизации
- Cache-friendly структуры

#### **6. [[Применение BST]]**  
*Практические задачи и применения бинарных деревьев поиска.*  
- Реализация множеств и словарей
- Range queries (запросы по диапазону)
- Статистические операции
- Геометрические применения
- Базы данных и файловые системы
